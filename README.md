# ECK stack Helm Chart

Requirements:
 1. Helm 3
 2. Kubernetes config file configured to the cluster

For a fast & easy installation run:

`./install.sh -n logging -p YOURPWD`


To uninstall the stack, run:
`./uninstall.sh -n logging`
